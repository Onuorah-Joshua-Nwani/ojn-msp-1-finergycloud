Usage Guidelines

✅ Do

  • Use the provided logo files
  • Maintain proper spacing around the logo
  • Use on contrasting backgrounds
  • Scale proportionally
  • Use high-resolution versions for print

❌ Don't

  • Modify the logo colors
  • Stretch or distort the logo
  • Add effects or shadows
  • Use on busy backgrounds
  • Place too close to other elements

⚠️ Important Notes

  • Clear Space: Always maintain a clear space around the logo equal to the height of the "F" in FinergyCloud
  • Minimum Sizes: Never use the logo smaller than the specified minimum sizes to ensure readability
  • File Formats: Use SVG for web and digital applications, PNG for social media, and vector formats for print
  • Background Contrast: Ensure sufficient contrast between the logo and background for accessibility

Logo Symbolism

Design Elements Meaning:

  • Cloud Shape: Represents cloud computing and digital transformation
  • Wind Turbine: Symbolizes renewable energy and sustainability
  • Data Bars: Represents AI analytics and data intelligence
  • Gradient Colors: Reflects innovation and environmental consciousness
  • Circular Design: Suggests completeness and global reach
